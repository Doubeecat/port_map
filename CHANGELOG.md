# 更新日志

## 版本 1.0.1 (2025-11-09)

### 修复
- 修复了 `check_existing_mapping` 函数中的 NoneType 错误
- 添加了对空输出的检查，防止 `split()` 方法在 None 对象上调用
- 修复了 GBK 编码错误，添加 `errors='ignore'` 参数处理特殊字符
- 改进了错误处理逻辑，确保即使检查映射失败也能正常显示状态

### 改进
- 增强了异常处理，程序在出错时会自动设置为"未映射"状态
- 优化了日志输出，提供更清晰的错误信息

## 版本 1.0.0 (2025-11-09)

### 功能
- 初始版本发布
- 实现端口映射GUI界面
- 支持创建/删除端口映射
- 自动检测现有映射
- IP地址和端口验证
- 管理员权限检查
